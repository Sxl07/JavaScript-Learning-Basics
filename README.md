## Español

## Descripción

**JavaScript Basics Learning** es un repositorio de práctica creado durante el proceso de aprendizaje de JavaScript. Su objetivo principal es reunir ejercicios básicos enfocados en comprender la sintaxis del lenguaje y reforzar los fundamentos necesarios para avanzar hacia proyectos web más completos.

Este repositorio no está pensado como una aplicación final, sino como un espacio académico y personal de entrenamiento donde se practican conceptos esenciales de programación con JavaScript.

## Objetivo del repositorio

El propósito de este proyecto es mejorar el dominio de JavaScript mediante ejercicios sencillos y progresivos relacionados con la sintaxis, la lógica de programación y el uso básico del lenguaje.

## Temas practicados

En este repositorio se pueden encontrar ejercicios orientados a conceptos como:

- Sintaxis básica de JavaScript
- Variables y tipos de datos
- Operadores
- Condicionales
- Ciclos
- Funciones
- Arreglos
- Objetos
- Manipulación básica de datos
- Práctica de lógica de programación

## Tecnologías utilizadas

- JavaScript
- HTML, si algunos ejercicios se ejecutan desde el navegador
- Navegador web
- Editor de código, como Visual Studio Code

## Cómo usar este repositorio

1. Clonar el repositorio:

```bash
git clone https://github.com/Sxl07/JavaScript.git
```

2. Entrar al proyecto:

```bash
cd JavaScript
```

3. Abrir los archivos en un editor de código.

4. Ejecutar los ejercicios según el tipo de archivo:

- Si son archivos `.js`, pueden ejecutarse con Node.js:

```bash
node nombre-del-archivo.js
```

- Si están asociados a archivos HTML, pueden abrirse directamente en el navegador.

## Estado del proyecto

Este repositorio corresponde a una etapa inicial de aprendizaje. Su contenido está orientado a reforzar fundamentos y practicar la sintaxis base del lenguaje JavaScript.

## Notas

Este proyecto forma parte de mi proceso de formación como estudiante de Ingeniería de Sistemas y representa una base para continuar avanzando en desarrollo web, frontend, backend con Node.js y proyectos full stack.

## English

## Description

**JavaScript Basics Learning** is a practice repository created during the JavaScript learning process. Its main goal is to collect basic exercises focused on understanding the language syntax and strengthening the core foundations needed to move toward more complete web development projects.

This repository is not intended to be a final application. Instead, it works as an academic and personal training space to practice essential programming concepts with JavaScript.

## Repository goal

The purpose of this project is to improve JavaScript proficiency through simple and progressive exercises related to syntax, programming logic, and basic language usage.

## Practiced topics

This repository may include exercises focused on concepts such as:

- Basic JavaScript syntax
- Variables and data types
- Operators
- Conditionals
- Loops
- Functions
- Arrays
- Objects
- Basic data manipulation
- Programming logic practice

## Technologies used

- JavaScript
- HTML, if some exercises are executed in the browser
- Web browser
- Code editor, such as Visual Studio Code

## How to use this repository

1. Clone the repository:

```bash
git clone https://github.com/Sxl07/JavaScript.git
```

2. Enter the project folder:

```bash
cd JavaScript
```

3. Open the files in a code editor.

4. Run the exercises depending on the file type:

- If they are `.js` files, they can be executed with Node.js:

```bash
node file-name.js
```

- If they are connected to HTML files, they can be opened directly in the browser.

## Project status

This repository belongs to an early learning stage. Its content is focused on reinforcing fundamentals and practicing the basic syntax of the JavaScript language.
